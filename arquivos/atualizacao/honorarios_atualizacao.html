Manual Interativo de Cálculos Trabalhistas

Um guia completo e interativo para entender os cálculos na esfera judicial trabalhista.

Lançamento e Atualização de Honorários

O módulo de Honorários (acessado via Atualização) permite o lançamento detalhado de verbas honorárias (Advocatícios, Periciais, etc.), aplicando o percentual devido sobre a base de cálculo definida na decisão judicial. É essencial cadastrar esses valores para o correto cálculo de atualização do passivo.

1. Inclusão de Novo Honorário

A tela inicial de Honorários, após a atualização, mostra o painel e o botão "Novo" para a inclusão de um item.

2. Configuração dos Dados Principais

Ao clicar em "Novo", abre-se a tela de cadastro. O primeiro passo é definir o Tipo de Honorário (Advocatícios, Periciais), o Devedor (Reclamante ou Reclamado) e a Descrição (ex: HONORÁRIOS ADVOCATÍCIOS).

3. Tipo de Valor e Base para Apuração

* Tipo do Valor: Marque "Calculado" para que o sistema use a alíquota e a base para apuração. * Base para Apuração: Selecione a base de cálculo determinada na sentença (ex: Valor da Liquidação, Valor do Bem, etc.). * Alíquota (%) e Data do Evento: Preencha o percentual e a data a partir da qual o valor deve ser atualizado. * Credor: Preencha o nome completo e o documento fiscal (CPF ou CNPJ) do credor dos honorários.

Clique em "Salvar" para registrar o honorário e, depois, em "Liquidar Atualização" no menu lateral para incluir os valores atualizados no cálculo final.
